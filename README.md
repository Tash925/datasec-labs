# DataSec Labs

Interactive cybersecurity learning experiences built alongside DataSec Chronicles.

## About

DataSec Labs is the hands-on side of the DataSec ecosystem.

While DataSec Chronicles focuses on articles, tutorials, and companion guides, DataSec Labs provides interactive learning experiences that help learners practice cybersecurity concepts through guided activities.
